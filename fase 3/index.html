Propósito:

El propósito de este sitio es el aprendizaje sobre los elementos y etiquetas basicas de un archivo Html, enseñar que los elementos y las etiquetas a pesar de complementarse no son la misma cosas así dar paso a saber todas la etiquetas y valores que varian para dar una mejor atención.

Etiquetas Básicas de HTML

Las etiquetas de HTML son principalmente markup language, quiere decir que puede ser leido sin haberlo compilado,
estas etiquetas o lenguaje tienen el proposito de dar instrucciones al navegador web de como presentar el texto al publico estas etiquetas tienen partes especificas que son:
-Carácter Menor qué.
-Palabra que determina a la etiqueta.
-Cualquier número de atriburos que se quiera usar.
-El carácter mayor qué.

Atributos y Valores

Los atributos y valores de HTML son información adicional de las etiquetas para personalizarlas, no todos los atributos se pueden utilizar en todas las etiquetas cada etiqueta tiene su propia lista de atributos que se pueden usar, además en cada atributo indica el tipo de valor que se puede asignar si el valor no es valido la información se omite.
